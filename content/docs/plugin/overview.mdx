# CalibrateDS Figma Plugin

While the CalibrateDS CLI handles code generation, the **CalibrateDS Figma Plugin** is designed specifically for Designers and Design Engineers working inside Figma. 

It is a complete quality toolkit providing accessibility auditing, component readiness checks, AI-powered stress testing, and multi-format token exporting—all without ever leaving Figma.

## 4 Modules, 1 Plugin

Most design quality workflows require stringing together multiple single-purpose plugins. CalibrateDS combines the entire pipeline into one interface:

1. **[Accessibility Advisor](/docs/plugin/accessibility-advisor)**: Context-aware WCAG auditing with inline fix suggestions.
2. **[Component Readiness](/docs/plugin/component-readiness)**: Validates if a component is structurally shippable to engineering.
3. **[Stress Test (AI)](/docs/plugin/stress-test)**: Simulates real-world content edge cases non-destructively.
4. **[Token Exporter](/docs/plugin/token-exporter)**: Export Figma local variable collections to 6 engineering-ready formats.

## Privacy & Local-First

CalibrateDS is built on a strict privacy model:
- **100% Local**: Zero cloud dependency. Your designs never leave your machine.
- **BYOK AI**: Bring Your Own Key. If you use the AI-enhanced features, you connect your own API key (OpenAI, Anthropic, Gemini, Groq). Data goes straight to your provider.
- **No Telemetry**: No analytics or tracking.
