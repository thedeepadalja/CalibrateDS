# Installation

**Requirements:** Node.js ≥ 20 — [download here](https://nodejs.org) if you don't have it.

[**View on npm: @calibrate-ds/cli →**](https://www.npmjs.com/package/@calibrate-ds/cli)

## Mac / Linux

```bash
npm install -g @calibrate-ds/cli
ptb --version
```

> If you get a permissions error, the recommended fix is to use [nvm](https://github.com/nvm-sh/nvm) to manage your Node installation — it installs global packages without requiring `sudo`. Alternatively, run `sudo npm install -g @calibrate-ds/cli`.

## Windows

Open **PowerShell as Administrator** and run:

```powershell
npm install -g @calibrate-ds/cli
ptb --version
```

## Updating

```bash
npm install -g @calibrate-ds/cli
```

After updating, **restart your AI IDE** so it re-spawns the MCP server with the new binary. You will see `[PTB MCP] vX.X.X started` in your IDE's MCP logs to confirm which version is active.
