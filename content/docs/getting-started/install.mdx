# Installation

**Requirements:** Node.js ≥ 20 — [download here](https://nodejs.org) if you don't have it.

## Mac / Linux

```bash
npm install -g @calibrate-ds/cli
ptb --version
```

> If you get a permissions error, run `sudo npm install -g @calibrate-ds/cli` or configure a [user-level npm prefix](https://docs.npmjs.com/resolving-eacces-permissions-errors-when-installing-packages-globally).

## Windows

Open **PowerShell as Administrator** and run:

```powershell
npm install -g @calibrate-ds/cli
ptb --version
```

## Updating

```bash
npm install -g @calibrate-ds/cli
```

After updating, **restart your AI IDE** so it re-spawns the MCP server with the new binary. You will see `[PTB MCP] vX.X.X started` in your IDE's MCP logs to confirm which version is active.
